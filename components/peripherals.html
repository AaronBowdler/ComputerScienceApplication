Peripherals

A peripheral device is "an ancillary device used to put information into and get information out of the computer.” Two categories of peripheral devices exist based on their relationship with the computer: an input device sends data or instructions to the computer, such as a mouse, keyboard, graphics tablet, image scanner, barcode reader, game controller, light pen, light gun, microphone, digital camera, webcam, dance pad, and read-only memory an output device provides output from the computer, such as a computer monitor, projector, printer, headphones, and computer speaker Many modern electronic devices, such as digital watches, smartphones, and tablet computers, have interfaces that allow them to be used as computer peripheral devices. Cloud storage involves stashing data on hardware in a remote physical location, which can be accessed from any device via the internet. Clients send files to a data server maintained by a cloud provider instead of (or as well as) storing it on their own hard drives. Some of the dangers of storage clouds are that from security holes to support issues, below are eight risks all users take when migrating to and storing their data in the cloud.
